## robotinterface - step 2

At first, compile launch.cc:
~~~
mkdir build
cd build
cmake ..
make
~~~

- 1st terminal:
  ~~~
  yarp server
  ~~~
- 2nd terminal:
  ~~~
  ./build/launch
  ~~~
- 3rd terminal:
  ~~~
  yarp name list
  yarp read /read /prova/measures:o
